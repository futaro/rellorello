## install

```bash
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ npm install
$ touch database/database.sqlite
$ php artisan migrate:refresh --seed
```

## run php server
```bash
$ php artisan serve
```

## run webpack(watch)
```bash
$ npm run watch
```
